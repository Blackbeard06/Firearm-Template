# Firearm-Template

What does this mod do?
This mod does nothing! It is a template for new modders to use when adding new firearms. For use with How to make a firearm mod build 41 54
Do not forget to change the place holder names and the mod id!

Features:
Contains place holders and organized code for adding new firearms.

Compatibility
This Template is for Build version 41.54 and beyond!
